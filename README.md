### FAAAH
Plays sounds when tests pass or fail.

### Features
- Plays different sounds for test success and failure.
- Set volume and choose sounds independently for both.
- Use built-in sounds or pick your own `.wav` file.
- Verify sounds in the settings.

### Actions (Tools Menu)
- **Configure Test Sounds...**: Settings for volume and files.
- **Enable Test Failure Sound**: Toggle failure sound.
- **Enable Test Success Sound**: Toggle success sound.
